## Notas

- Las dimensiones disponibles en Commons son las máximas que la plataforma
  sirve para cada archivo. Si el municipio entrega versiones de mayor
  resolución (entrega oficial), sustituir el archivo manteniendo el mismo
  nombre para no romper las referencias en el código.
- Para uso público con atribución estricta, citar autor + URL de la página
  de archivo en Commons. Los autores aparecen en la página de archivo de
  cada imagen.
